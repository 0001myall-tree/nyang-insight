# Nyang Insight

Nyang Insight는 흩어진 지식을 모아 나만의 아카이브로 정리하고, AI와 함께 새로운 인사이트를 연결해보는 개인 지식 공간입니다.

## 실행 방법

1. 이 저장소를 다운로드하거나 ZIP으로 내려받아 압축을 풉니다.
2. Mac은 `실행.command` 파일을 더블클릭합니다.
3. Windows는 `실행(윈도우).bat` 파일을 더블클릭합니다.
4. 브라우저에서 `http://localhost:8000`을 엽니다.

## 직접 실행

```bash
python3 server.py
```

서버가 실행되면 브라우저에서 `http://localhost:8000`으로 접속하세요.

## 참고

`notes.json`은 개인 노트가 저장되는 파일입니다. 배포용 저장소에는 빈 파일로 포함되어 있습니다.
